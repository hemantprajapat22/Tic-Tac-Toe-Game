# 🎮 Tic-Tac-Toe Game (Python)

A simple **Tic-Tac-Toe** game built using **Python**.  
Play this classic two-player game right in your terminal! 🚀

---

## 📌 Features

✅ Playable in **command-line interface (CLI)**  
✅ **Two-player** game (X vs O)  
✅ **3x3 grid** with proper input validation  
✅ **Win detection** & **draw handling**  
✅ Simple, lightweight & beginner-friendly code

---

## 🛠️ Tech Stack

- **Language**: Python 🐍
- **IDE Support**: VS Code, PyCharm, Jupyter, etc.
- **Platform**: Cross-platform (Windows, Mac, Linux)

🎮 Game Rules

The game is played on a 3x3 grid.

Player X goes first, followed by Player O.

Players take turns marking spaces in the grid.

The first player to align 3 marks horizontally, vertically, or diagonally wins.

If all 9 cells are filled and no one wins, the result is a draw.
